# 💫 About Me:
I'm a full stack developer with a passion for building polished web and mobile experiences end-to-end. I work across the entire stack — from crafting clean, responsive interfaces with React and Next.js to designing robust backends with Node.js, Express, and Supabase. On the mobile side, React Native lets me ship cross-platform apps without compromising on feel.<br>I write primarily in TypeScript and Python, and I'm comfortable working with SQL to design and query databases. Beyond dev, I'm a cybersecurity enthusiast and active CTF player — I love the puzzle-solving mindset that hacking brings, and it keeps my engineering sharp and security-conscious.
